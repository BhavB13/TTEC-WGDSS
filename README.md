# TTEC-WGDSS

## Trinidad and Tobago Weather Grid Decision Support System

TTEC-WGDSS is a decision support system designed to assist Control Engineers with weather-aware generation and grid operation decisions for Trinidad and Tobago’s national electrical network.

The system combines weather data, forecast information, generation data, and operational decision logic to help engineers better assess potential grid risks and make informed recommendations.

---

## Project Purpose

Weather conditions can affect electricity demand, generation availability, transmission stability, and overall grid reliability. This project aims to support grid operators by providing a centralized platform for monitoring weather conditions and evaluating their potential impact on generation decisions.

---

## Key Objectives

* Monitor current and forecasted weather conditions across Trinidad and Tobago.
* Support Control Engineers in making weather-based generation decisions.
* Provide decision recommendations based on operational and environmental data.
* Build a foundation for future SCADA data integration.
* Improve visibility into weather-related grid risks.

---

## Planned Features

* Weather dashboard for Trinidad and Tobago.
* Interactive map with weather overlays.
* Generation and demand monitoring.
* Forecast-based risk scoring.
* Decision recommendation engine.
* Historical data storage and analysis.
* Future machine learning support for demand forecasting.
* Future integration with SCADA and live operational systems.

## System Overview

The application is designed as a modular platform with separate services for:

* Weather data collection
* Forecast processing
* Generation data handling
* Risk scoring
* Recommendation logic
* API communication
* Frontend visualization

This structure allows the system to begin with mock operational data and later expand into real-time utility data sources.

---

## Current Development Status

This project is currently under development. The first version focuses on building the foundation of the platform, including the backend API, frontend dashboard, weather data integration, and mock generation data.

Future versions will focus on improving forecasting accuracy, adding machine learning models, and integrating real operational data.

---

## Long-Term Vision

The long-term goal of TTEC-WGDSS is to become a practical decision support tool for weather-based power system operations in Trinidad and Tobago. The system should help engineers identify risks earlier, improve generation planning, and support more reliable grid operations.

---

## Disclaimer

This project is currently a development prototype and is not connected to live T&TEC operational systems. Any recommendations generated by the system should be treated as informational until validated by qualified power system engineers and real operational data.
